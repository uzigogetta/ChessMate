## How to run
- Install deps: `pnpm install`
- Start (Dev Client): `npx expo start --dev-client`
- Lint: `pnpm lint`
- Format: `pnpm format`

## Env
- Set `EXPO_PUBLIC_SENTRY_DSN` in `.env` or your variables provider when ready.
